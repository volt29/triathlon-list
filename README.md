# Triathlony w Europie 2026

Statyczna tabela HTML z zawodami triathlonowymi w Europie w okresie czerwiec-październik 2026.

Zakres:

- pełny dystans,
- dystans 1/2,
- cena pakietu, jeśli była publicznie dostępna,
- dystans autem z Warszawy do miejsca startu.

Strona wejściowa `index.html` przekierowuje do głównej tabeli:
`triathlon_europe_jun-oct_2026_full_half.html`.
